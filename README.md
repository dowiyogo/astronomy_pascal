# Colección de programas de astronomía (BASIC / Pascal)

Este repositorio reúne índices y documentación (en Markdown) de colecciones clásicas de programas publicados/descritos en material de astronomía.

## Estructura del repositorio

### `astronomy/` — *Astronomy on the Personal Computer*
Contiene el README (convertido desde `AAREADME.DOC`) asociado al libro **“Astronomy on the Personal Computer (2nd edition, 1994)”** de O. Montenbruck y T. Pfleger.  
Incluye el listado de programas en **Pascal**, su organización por subdirectorios (p. ej. `PLANETS/`, `MOON/`, `TIME/`, `UNITS/`, etc.) y una guía breve de compilación/uso.

### `computing/` — *Astronomical Computing (Sky & Telescope)*
Contiene el README (convertido desde `INDEX.TXT`) con el **índice de programas** publicados en la sección *Astronomical Computing* de **Sky & Telescope**.  
Incluye una tabla con **nombre de archivo**, **edición/página** y una **descripción breve** de cada programa.
